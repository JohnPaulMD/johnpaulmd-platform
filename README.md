# 🚀 JohnPaulMD Platform

> Professional portfolio platform showcasing data analytics, research, and AI innovation.

---

## 🌟 Overview

JohnPaulMD Platform is the official digital platform of **JohnPaul Ozoigbo**, built under **Manus Dei Solutions**.

The platform is designed to showcase:

- 📊 Data Analytics Portfolio
- 📚 Research & Statistical Analysis
- 🚀 Innovation Lab
- 🧠 MedLearn AI
- 📖 Professional Insights
- ⭐ Client Reviews
- 📄 Interactive CV
- 🔐 Secure Admin Dashboard

---

## 🛠️ Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
- Supabase
- PostgreSQL
- Vercel

---

## 🎯 Vision

To build a premium digital platform that demonstrates how data, research, and intelligent software can solve real-world problems.
